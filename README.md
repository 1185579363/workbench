# 工具导航工作台

这是一个基于 `index.html` 的纯静态工具导航页，可直接部署到 GitHub Pages。页面内置常用交易工具、AI 工具和日常工具入口，同时支持搜索、分类筛选、收藏、自定义工具管理，以及 JSON 导入导出。

## 在线说明

- 入口文件：`index.html`
- 技术栈：原生 HTML + CSS + JavaScript
- 数据存储：浏览器 `localStorage`
- 部署方式：静态托管即可，适合 GitHub Pages

## 主要功能

- 工具卡片导航，集中展示常用站点和本地工具入口
- 支持按分类、来源进行筛选
- 支持关键词搜索名称、描述、分类和来源
- 支持浅色 / 深色主题切换
- 支持收藏工具并置顶显示
- 支持新增、编辑、删除自定义工具
- 支持分类管理，包含新增、重命名、删除
- 支持导出当前配置为 JSON
- 支持通过 JSON 导入并恢复工具数据

## 本地使用

1. 下载或克隆仓库到本地。
2. 直接用浏览器打开 `index.html`。
3. 如需完整使用本地工具入口，请确保以下文件和目录与 `index.html` 保持同级或原始目录结构：

- `年化收益率计算器.html`
- `合约计算器.html`
- `货币转换.html`
- `电费计算器.html`
- `倒计时.html`
- `tts.html`
- `favicon.svg`
- `安全密码生成器/`

## 使用方法

- 在顶部搜索框中输入关键词，可快速过滤工具
- 点击分类或来源标签，可切换筛选范围
- 点击卡片右上角星标，可收藏或取消收藏
- 点击“添加工具”可新增自定义工具
- 点击“分类管理”可维护自己的分类
- 点击“导出 JSON”可备份当前配置
- 点击“导入 JSON”可恢复之前导出的配置
- 右键工具卡片可进行编辑或删除

## 注意事项

- 这是纯前端项目，不需要安装依赖，也不需要后端服务。
- 页面中的个性化数据保存在当前浏览器的 `localStorage` 中，换浏览器或清除缓存后不会自动同步。

## 适用场景

- 个人工具导航页
- GitHub Pages 静态主页
- 常用网站和本地页面入口聚合
- 可自行扩展的轻量工具箱首页
