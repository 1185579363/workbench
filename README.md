# 工具导航工作台

这是一个可直接部署到 GitHub Pages 的静态站点。

## 目录说明

- `index.html`：导航首页
- `年化收益率计算器.html`、`合约计算器.html`、`货币转换.html`、`电费计算器.html`、`倒计时.html`：本地工具页
- `安全密码生成器/`：密码生成器页面资源

## 部署到 GitHub Pages

1. 在 GitHub 新建一个公开仓库。
2. 把当前目录全部上传到仓库根目录。
3. 进入仓库 `Settings` -> `Pages`。
4. 在 `Build and deployment` 中选择 `Deploy from a branch`。
5. 选择 `main` 分支和 `/ (root)` 目录，保存。
6. 等待 GitHub 完成发布。

发布成功后，访问地址通常是：

- `https://你的用户名.github.io/仓库名/`

## 本地链接规则

这个项目已经把首页中的本地工具入口改成了相对路径，适合网页部署。

例如：

- `./年化收益率计算器.html`
- `./安全密码生成器/`

后续如果你新增工具，优先使用：

- 外链：`https://example.com`
- 仓库内页面：`./你的页面.html`
- 仓库内文件夹页面：`./你的文件夹/`
